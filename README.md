# Кредитный рейтинг компаний (РСБУ)

Проект читает бухгалтерскую отчётность (БФО) в формате Excel, рассчитывает
кредитный рейтинг компаний по взвешенной балльной модели и показывает
результат на дашборде со «светофором».

Методика полностью описана в [`docs/METHODOLOGY.md`](docs/METHODOLOGY.md).

## Структура

| Папка | Назначение |
|---|---|
| `data/input/` | **Входная отчётность** — сюда кладём файлы БФО (`.xlsx`) |
| `data/output/` | Результаты расчёта (`results.json`) |
| `credit_analysis/` | Ядро методики: чтение, коэффициенты, рейтинг, выводы |
| `dashboard/` | Генератор статического HTML-дашборда с кредитным рейтингом компаний |
| `docs/` | Методология расчёта |

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# 1) Положите файлы БФО (.xlsx) в data/input/
#    (в комплекте есть реальный пример RUSSKII-KVARTs.xlsx — выгрузка ГИР БО)

# 2) Прогон из командной строки:
python -m credit_analysis.pipeline data/input

# 3) Дашборд (статический HTML, без сервера):
python dashboard/html_report.py
```

## Формат входного файла

`.xlsx` с тремя листами: «Сведения об организации», «Бухгалтерский баланс»,
«Отчет о финансовых результатах» (лишние листы, напр. ОДДС, игнорируются).
Поддерживается штатный формат выгрузки **ГИР БО** (государственный реестр
бухгалтерской отчётности). Строки отчётности определяются по кодам
РСБУ (1200, 1210, …, 2110, 2400), пустые колонки-разделители
автоматически отбрасываются. Периоды читаются слева направо: текущий →
предыдущий → позапрошлый год.

## Что считает модель

1. **11 коэффициентов** (ликвидность, автономия, долговая нагрузка, ICR,
   оборачиваемость, ROE/ROA, рентабельность) + чистая прибыль.
2. **Балльная оценка «светофор»**: 0 / 0,5 / 1 балл по двум порогам.
3. **Комплексный балл 0–100** — взвешенная сумма 6 ключевых факторов.
4. **Класс кредитоспособности**: A (≥70), B (50–69), C (30–49), D (<30).
5. **Качественные выводы**: динамика, ликвидность, долг, покрытие процентов,
   качество прибыли, признак холдинговой модели.

## Тесты

```bash
pip install -r requirements-dev.txt
pytest
```

Набор `tests/` покрывает пороги и балльную оценку (`metrics.py`), веса и
классы кредитоспособности (`rating.py`), нормализацию чисел и чтение по
кодам РСБУ (`extract.py`), качественные выводы (`conclusions.py`) и сквозной
прогон пайплайна (`pipeline.py`). Дополнительно к этому в `metrics.py` и
`rating.py` остаются встроенные self-check блоки в `if __name__ == "__main__"`.

## Дашборд

```bash
python dashboard/html_report.py            # data/input -> data/output/report.html
python dashboard/html_report.py data/input data/output/report.html  # явные пути
```

Расчётная логика полностью в `credit_analysis/`, `dashboard/html_report.py`
— только представление. Результат — самодостаточный `.html`-файл (инлайн
CSS/JS, без внешних запросов и интернета, без сервера) — можно открыть
прямо в браузере (`open data/output/report.html`) или выложить как
статику:

- Сводная таблица компаний с классом и баллом (текущий/предыдущий год).
- Распределение по классам A/B/C/D.
- Диаграмма комплексного балла по компаниям.
- Детальный «светофор» по всем показателям выбранной компании (переключение кликом).
- Текстовые выводы.
- Переключатель светлой/тёмной темы.

## Ограничения

Модель рассчитана на нефинансовые организации по РСБУ. Пороги — общеотраслевые,
веса заданы экспертно (не откалиброваны на дефолтах). Подробнее — раздел 7
методологии.
